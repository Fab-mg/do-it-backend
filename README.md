# Do-it-backend

## Petite notice pour lancer le projet.

## Installation

Apres avoir cloné le projet, naviguez vers le repertoire du projet: cd do-it

Installer les dépendances avec: npm install

Creer votre propre fichier ".env" basé sur le contenu de "example.env" et remplacez les valeurs par les votre.

Lancer le serveur: npm run start
